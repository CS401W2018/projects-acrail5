What is the difference between block tags and inline tags?

block tags always start on a new line and take up the full width available. and inline tags are Do not start on a new line and only take up as much width as necessary. Common examples: span, a, strong, em, img, button.

What is the margin, border, and padding?

Margin: The space outside an element’s border, separating it from other elements. Border: The boundary around an element, which can have a color, width, and style. Padding: The space inside an element’s border, creating space between the content and the border.

What are units of measurement you can use in CSS?

Absolute Units (Fixed Size) px (pixels) – Most commonly used. pt (points) – Used in print media. cm, mm, in – Rarely used in web design. Relative Units (Flexible & Responsive) % – Percentage of the parent element. em – Relative to the font size of the element. rem – Relative to the root (html) font size. vw, vh – Viewport width and height. vmin, vmax – Based on the smallest/largest viewport dimension.
